# :pushpin: Descrição do Projeto <a name="id01"></a>
Este projeto tem como objetivo automatizar testes funcionais da aplicação ServeRest utilizando Selenium WebDriver com JavaScript (Node.js).

A automação foi desenvolvida seguindo boas práticas de organização de código, utilizando o padrão Page Object Model (POM), visando facilitar a manutenção, reutilização e escalabilidade dos testes.

# 🎯 Desafio  <a name="id04"></a>
Automatizar os principais fluxos da aplicação ServeRest:

Acesso à tela de Login
Autenticação de usuários
Validação de mensagens e comportamentos da aplicação
Navegação entre páginas
Execução automatizada dos cenários de teste

O objetivo é garantir a qualidade da aplicação através da validação automática dos fluxos mais importantes do sistema.

# 🗂 Instalações <a name="id03"></a>
#### Baixar Projeto e instalações
```bash
    # Clonar o repositório
    $ git clone https://github.com/Amanda92Cortez/selenium-serverest.git

    # Entrar no diretório
    $ cd selenium-serverest

    # Instalar as dependências
    $ npm install

    # Ou instalar separado
    $ npm install selenium-webdriver 
    $ npm install chromedriver
```

#### ❎ Rodar teste especifico ou todos

```bash
    $ node tests/login.test.js
    ou
    $ npm test
```
